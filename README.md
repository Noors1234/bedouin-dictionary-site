# Bedouin Dictionary

This project is a **Bedouin Dictionary Website**. It aims to provide users with an interactive platform to explore Bedouin words, phrases, and pronunciations. The website is designed to be simple, responsive, and ready for deployment.

## Features

- Browse Bedouin words and phrases
- Listen to audio pronunciations
- Simple, clean interface
- Easily deployable to hosting platforms like Render
